Admin Dashboard

A modern and responsive Admin Dashboard UI built using HTML, CSS, and JavaScript, showcasing analytics, user activity, and recent orders in a clean layout.

Live Link

Project Overview

This Admin Dashboard is a responsive frontend design built with HTML, CSS, and JavaScript. It features an elegant dark-themed interface with a sidebar navigation, analytics cards, new user highlights, recent orders, and a reminder section. The layout is clean, modern, and optimized for usability, making it a great template for any admin or data visualization interface.

The project focuses purely on frontend development, showcasing design skills, layout structuring, and UI components without backend functionality.
